# arm_analyzer
```
+---------------------+
|  输入：C/C++ 源代码 |
+---------------------+
          |
          v
+-----------------------------+
| LLVM LibTooling AST Visitor| ← 解析并抽象为语法树
+-----------------------------+
          |
          v
+---------------------------+
|  IR/AST 转换为关系数据    | ← 提取需要的信息（如变量名、操作类型、数据类型、循环结构等）
+---------------------------+
          |
          v
+------------------------+
|    Soufflé Datalog     | ← 用规则匹配优化模式
+------------------------+
          |
          v
+-----------------------------+
| 匹配结果 + 建议替换方式    | ← 输出匹配信息和建议替换代码
+-----------------------------+
```
